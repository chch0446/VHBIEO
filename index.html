Class: dnas:Actions   back to top

NOTE dnas:Actions is a leaf node.

Description

Actions are interactions with systems or activities that an occupant can conduct in order to satisfy their needs. The violation of one or more of an occupant's needs leads to discomfort. There- fore, this uncomfortable state for the occupant will provoke an action.
The action may be an interaction with a system in which the occupant conjectures that their action will restore comfort. - This class is associated with "State". - Input an action ID,assign label, and give description in comment. For example, Action0001 labels as turning on the light. Moreover, it can be multiplt action such as turning on the light + openning window.
